# Resonance

**Cinematic visual generation tool for organic and ambient music.**

Create unique, narrative-driven visuals that respond to the subtle nuances of real instruments - breath, sustain, vibrato, silence, and emotional arc.

## Tech Stack

| Layer | Technology |
|-------|------------|
| Frontend | Next.js 14 (App Router) + TypeScript |
| Styling | Tailwind CSS + shadcn/ui |
| Database | Supabase (PostgreSQL) |
| Hosting | Vercel |
| Auth | Supabase Auth |
| Payments | Stripe |
| Audio Analysis | Essentia.js + Meyda + Web Audio API |
| Video Rendering | FFmpeg + Canvas + WebGL |
| AI Orchestration | Claude (Sonnet 4) |
| Worker | Railway/Render |
| Storage | Cloudflare R2 |
